# Webpack SPA ( pug + stylus ) dev server

## setup

```sh
$ yarn install
```

## dev server

open new browser & start webpack dev server (HMR)
```sh
$ yarn run start
```

start webpack dev server (HMR) without open browser
```sh
$ yarn run restart
```

## build

production mode
```sh
$ yarn run build
```

development mode
```sh
$ yarn run build:dev
```
